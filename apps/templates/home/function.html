Function

Automatic Grading

자동채점 프로그램

"채점의 끝"의 대표적인 기능으로, 영어 주관식 서술형을 자동으로 채점해주는 서비스입니다.
자동채점하고 싶은 답안 파일과 학생들의 답안 파일을 업로드해주시면 자동으로 채점 및 학생별로 점수 계산이 됩니다.
자동채점 서비스를 이용하고 싶으시다면, Function > Automatic Grading을 눌러주세요!

Storage

채.끝 저장소

자동채점 기능을 사용하고, 업로드한 파일과 프로그램에 돌린 파일 등을 모두 저장해주는 채.끝 저장소 서비스입니다.
채.끝 저장소에 채점에 필요한 데이터를 저장하여 정리하고, 편리하게 필요할 때마다 찾아서 사용해보세요!
DB 서비스를 이용하고 싶으시다면, Function > Storage을 눌러주세요!
